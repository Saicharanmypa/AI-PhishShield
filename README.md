# AI-PhishShield

AI powered system for detecting phishing websites using Machine Learning and XGBoost.

## Overview
The AI-PhishShield detects whether a URL is that of a phishing site or not using machine learning algorithms trained on both phishing and benign webpages.

The project is able to extract features from the URLs and their domains and make predictions using the XGBoost classifier.
